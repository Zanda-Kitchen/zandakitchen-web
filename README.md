# Zanda Kitchen — Marketing Site

This repository contains the public marketing website for [Zanda Kitchen](https://zandakitchen.com), built with Next.js and deployed on Vercel.

Zanda Kitchen is an upcoming iOS app for home cooks who want to improve over time. It lets you build recipes, track versions, rate each attempt, and refine your cooking until a dish is exactly right.

## Stack

- [Next.js](https://nextjs.org) (App Router)
- [Tailwind CSS](https://tailwindcss.com) v4
- [Framer Motion](https://www.framer.com/motion/)
- [Lucide React](https://lucide.dev)

## Development

```bash
npm install
npm run dev
```
